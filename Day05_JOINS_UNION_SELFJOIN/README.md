# Day 05 - SQL JOINs, UNION & SELF JOIN

## Topics Covered

- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- UNION
- UNION ALL
- Combining multiple columns
- Self JOIN using table aliases

## Key Concepts Learned

- JOIN combines tables using related keys.
- LEFT JOIN keeps all left table records.
- RIGHT JOIN keeps all right table records.
- UNION removes duplicates.
- UNION ALL keeps duplicates.
- Self JOIN is used when a table references itself.

This day focused on relational database concepts and real-world relationship modeling.
